# Shravan Parthe Portfolio

A Streamlit portfolio designed as a case study journal instead of a conventional resume page.

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy Free On Streamlit Community Cloud

1. Push this repository to GitHub.
2. Create a new app on Streamlit Community Cloud.
3. Set the main file path to `app.py`.
4. Use Python 3.11 or 3.12 if the platform asks for a version.

The app has one runtime dependency: Streamlit.
